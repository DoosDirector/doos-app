# Doo's - Project Plan

## Vision
Doo's is a simple, delightful web app (PWA) that helps workplace teams organise events and activities (nights out, lunches, team-building, coffee runs, etc.). It replaces chaotic Teams/Slack threads with polls, maps, RSVPs, and a shared Memory Box.

## Target Users
- Office workers in the UK (especially Scotland & England)
- Teams using Microsoft Teams / Slack

## MVP Success Criteria
- First 10 real events created and used by real teams within 2 weeks of launch
- Users can create, RSVP, view map, and add memories without friction

## Core MVP Features (in order)
1. Authentication (email + Google/Microsoft SSO)
2. Dashboard – list of my events + "Create new event"
3. Event creation (title, date, description, type, alcohol toggle/icon)
4. Polls inside event (venue/date/activity options)
5. Google Maps integration – pick venues, add stops, show route
6. RSVP with drinking preference (Yes/No/Maybe alcohol)
7. Memory Box – upload/view photos & videos per event (post-event)
8. Shareable event link + "Post to Teams/Slack" rich preview
9. Simple recommendations ("Suggested nearby activities")
10. One-click social sharing (LinkedIn/Instagram/Facebook) with deep link + memory teaser

## Non-MVP (explicitly out for v1)
- Native mobile apps
- Native Teams/Slack bots
- Payments / subscriptions
- Admin/company dashboards
- Public event discovery
- Advanced analytics

# Doo's - Detailed Requirements

## User Stories (MVP)
- As a user I can sign in with work Google/Microsoft account
- As a user I can create an event in < 60 seconds
- As a user I can add multiple poll questions (venue, date, activity)
- As a user I can pick venues on Google Maps and plan stops
- As a user I can RSVP and say whether I'm drinking alcohol
- As an attendee I can upload photos/videos to the Memory Box after the event
- As an organiser I can share a beautiful event card to Teams/Slack
- As a user I can share memories to social media with link back to Doo's

## Design Principles
- Mobile-first, clean, fun, professional (not childish)
- Fast loading, delightful animations
- UK-friendly (British English, timezone handling)
